## 衍生數據

`warning_series`、`bulletin_events`、來源連結及由政府原始資料正規化而成的匯出檔，
按照 [DATA.GOV.HK 使用條款](https://data.gov.hk/tc/terms-and-conditions) 提供。
使用或再發布時應：

- 清楚註明資料來源為「香港天文台」及／或「香港特別行政區政府新聞公報」；
- 附上相應 `source_url`，並保留本專案提供的取得時間及 checksum；
- 不暗示香港特別行政區政府認可本專案或其分析；
- 自行查閱並遵守發布當時的最新政府條款。

政府資料按其原有條款及「現狀」提供。本專案的解析、配對及分類可能有錯誤，
不應用作保障生命或財產的即時警告服務。

## 原始 HTML

`data/raw/` 是可重現研究存檔，不屬於 MIT 授權範圍。預設 Git repository 及
Open Data 套件只發布來源 URL、metadata、checksum 和衍生欄位；原始 HTML
另行保存。任何原文重新發布均須按來源網站當時適用的條款處理。
